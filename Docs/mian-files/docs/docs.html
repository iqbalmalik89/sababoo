Javascript and jQuery

This template imports several Javascript libraries and files, i have put a link to original authors on the file names (ended with .js) where you can find more documentation on each of them:

  • Main plugins - jquery-1.11.3.min.js
  • jQuery Migrate - jquery-migrate-1.2.1.min.js
  • Bootstrap's JavaScript components - bootstrap.min.js
  • jQuery Waypoints is the easiest way to trigger a function when you scroll to an element. - jquery.waypoints.min.js
  • jQuery Easing Plugin is a jQuery plugin from GSGD to give advanced easing options. - jquery.easing.1.3.js
  • jQuery SmoothScroll Plugin is experiencing for websites for mouse wheel, keyboard and touchpad scrolling- SmoothScroll.min.js
  • jQuery Wow Plugin is for revealing animations when you scroll. It's shoule be used with Animate.css - wow.min.js
  • jQuery SlickNav Plugin is responsive mobile menu - jquery.slicknav.min.js
  • jQuery Placeholder Plugin is plugin that enables HTML5 placeholder behavior for browsers - jquery.placeholder.min.js
  • Bootstrap Wysihtml5 is a simple, beautiful wysiwyg editors - bootstrap3-wysihtml5.min.js
  • jQuery IntroLoader Plugin is a jQuery plugin for generate animated intro loading pages - jquery.introLoader.min.js
  • Bootstrap-select is a jQuery plugin that utilizes Bootstrap's dropdown.js to style and bring additional functionality to standard select elements. - bootstrap-select.js
  • jQuery Responsivegrid Plugin is a plugin to make grid layout - jquery.responsivegrid.js
  • jQuery Slick Plugin is fully responsive carousel - slick.min.js
  • jQuery Validator Plugin is a user-friendly HTML5 form validation jQuery plugin for Bootstrap 3 - validator.min.js
  • Bootstrap-tokenfield is a jQuery tag/token input plugin for Twitter's Bootstrap. - bootstrap-tokenfield.js
  • jQuery InfoBox Plugin a plugin to customize google map infowindow - infobox.js
  • Bricklayer a lightweight and independent Pinterest-like cascading grid layout library. - bricklayer.js
  • jQuery - Easy Ticker plugin is a news ticker like plugin, which scrolls the list infinitely. It is highly customizable, flexible with lot of features and works in all browsers. Tags: slider, vertical, animation, effect, ticker, news, rss, feed, easing, horizontal - easy-ticker.js
  • Bootstrap File Input is an enhanced HTML 5 file input for Bootstrap 3.x with file preview for various files, offers multiple selection, and more - fileinput.min.js
  • Handlebars is an extension to the Mustache templating languag - handlebars.min.js
  • jQuery ion.rangeSlider Plugin is a range slider plugin - ion.rangeSlider.min.js
  • Animated counter is an animated counter - jquery.countimator.js
  • SheepIt! Plugin is a jQuery plugin that allows you to clone form elements dynamically - jquery.sheepItPlugin.js
  • jQuery Filestyle is a personalize your input file and let the beautiful - jquery-filestyle.min.js
  • Selectize is a hybrid of a textbox and select box. It's jQuery based and it has autocomplete and native-feeling keyboard navigation; useful for tagging, contact lists, etc. - selectize.min.js
  • typeahead - a fast and fully-featured autocomplete library typeahead.bundle.min.js
  • Main template scripts - you might edit in the file if you want to change to plugin options or settings
